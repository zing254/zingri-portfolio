# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-20

### Added

- **Project Infrastructure**
  - Next.js 15 with App Router setup
  - TypeScript 5.4 configuration
  - TailwindCSS 3.4 theming
  - Framer Motion 11 for animations
  - Three.js / React Three Fiber for 3D elements

- **Core Pages & Components**
  - Landing page with hero section
  - About section with bio and stats
  - Skills showcase with category filtering
  - Projects gallery with filtering
  - Experience timeline
  - Contact form with EmailJS integration

- **Configuration System**
  - Type-safe `src/lib/config.ts` for all content
  - Centralized theme configuration
  - Social links management
  - Project and experience data

- **Documentation**
  - README.md with cyberpunk aesthetic
  - Environment variables template (.env.example)
  - Contributing guidelines (CONTRIBUTING.md)
  - Changelog tracking (CHANGELOG.md)

### Features

- 🌑 Dark mode cyberpunk UI with cyan, purple, and neon green accents
- ⚡ Server-side rendering and optimized loading
- 🎨 Smooth page transitions and micro-interactions
- 📱 Fully responsive design
- 🔒 Environment-based configuration
- 🎭 3D animated elements
- 💾 Dynamic content management

### Technical

- **Languages:** TypeScript (strict mode)
- **Styling:** TailwindCSS with custom theme
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **3D:** Three.js / React Three Fiber
